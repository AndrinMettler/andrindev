Überblick über die verschiedenen CSS Selektoren:

Der "einfachste" Selektor ist der Element Selektor

Ein Element mit einer Klasse erhält mehr Gewicht

Ein Element mit einer ID erhält noch mehr Gewicht, da die ID spezifischer ist(kommt nur 1x vor)

Ein "Inline-Selektor" erhält den höchsten Wert und ist i.d.R der stärkste Selektor
